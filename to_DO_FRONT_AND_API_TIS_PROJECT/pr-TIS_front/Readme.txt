 1
- открываю  git CMD  правой кнопкой мышке в пустом месте файле 
- прописываю команды 

- - `npm i`
- `npm run start`- run api server nodemon
- получаем ссылку на локал сервер пример  http://localhost:3001/

- вставляю поэтапно пути приложения ( вставляем это в браузер ! ) 

- GET `/api/todos` - Получение всей коллекции дел
- GET `/api/todos/:id` - Получение коллекции под ID
- POST `/api/todos/` - Создание коллекции
  - Обязательно поля для создания дела:
    - "todo": 'Сдать практику', string
    - "chek": false, bool
- DELETE `/api/todos/:id` - удалении коллекции под ID
- PUT `/api/todos/:id` - Редактирование коллекции под ID
  - Обязательно поля для создания дела:
    - "todo": 'Сдать практику', string
    - "chek": false, bool 
- Делаем дизайн
- Оформляем html+json 
- Вот эту приблуду, можно через script формить :  {"success":true,"collection":[{"id":1,"todo":"Сдать практику","chek":false},{"id":2,"todo":"Получить оценку","chek":false},{"id":3,"todo":"Жить спокойно","chek":false}]}  Это мы получили при прописывании команды /api/todos
